JSON

JSON stands for JavaScript Object Notation.

JSON is a format for storing and transporting data.

JSON is often used when data is sent from a server to a web page.

Syntax Rules:

  • Data is in name:value pairs
  • Data is separated by commas
  • Curly braces hold objects
  • Square brackets hold arrays

Syntax:

 {
  "<name>" : "<value>",
  "<array_name>" : [value_1, value_2, ... value_n],
  "<object_name>" : [ { "<name>" : "<value>", .... }, ... ],
   ...
 }

Converting a JSON Text to a JavaScript Object

  • A common use of JSON is to read data from a web server, and display the data in a web page.
  • For simplicity, this can be demonstrated using a string as input.
  • First, create a JavaScript string containing JSON syntax.
  • Then, use the JavaScript built-in function JSON.parse() to convert the string into a JavaScript object.

Example:

 // JSON Object in String
 let jsonObj = '{"name": "Shiva","favorites": ["HTML","CSS", "JS"],' +
  ' "education":[{"degree":"M.Tech.", "university":"JNTUH"}, {"degree":"B.Tech.", "university":"JNTUH"}] '+
 '}';

 // Converting JSON Object to JavaScript Object
 const jsObj = JSON.parse(jsonObj);

 console.log(jsObj.name);

 console.log(jsObj.favorites);
 console.log(jsObj.favorites[0],  jsObj.favorites[1], jsObj.favorites[2]);

 console.log(jsObj.education);

 console.log(jsObj.education[0]);
 console.log(jsObj.education[0].degree,  jsObj.education[0].university);

 console.log(jsObj.education[1]);
 console.log(jsObj.education[1].degree,  jsObj.education[1].university);
