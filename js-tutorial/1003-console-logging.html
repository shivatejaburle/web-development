Console Logging

The console is useful for testing purposes.

The Console object provides access to the browser's debugging console.

console.log()

The console.log() method writes a message to the console.

Syntax

console.log(message);

Example: console.log("This is a log message.");

console.warn()

The console.warn() method writes a warning to the console.

Syntax

console.warn(message)

Example: console.warn("This is a warning!");

console.error()

The console.error() method writes an error message to the console.

Syntax

console.error(message)

Example: console.error("You made a mistake!!!");
