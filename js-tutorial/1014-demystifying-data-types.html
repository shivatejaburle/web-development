Demystifying Data Types

Demystifying means confusion, whenever you're working with someone code, you might not know what the variable type necessarily.

Variables can go through a lot of different states and, you know, maybe a variable is list at one point or an array.

Maybe it's an object at another point, maybe it's a string and at another point, maybe it's a number.

At some point, ideally that doesn't happen, but occasionally it does.

So you can always run a particular function called typeof().

Examples:

let var1 = "JavaScript";
console.log("var1 is " + typeof(var1));

let var2 = 1990;
console.log("var2 is " + typeof(var2));

let var3 = ["Item 1", "Item 2"];
console.log("var3 is " + typeof(var3));

let var4 = true;
console.log("var4 is " + typeof(var4));

Output:

var1 is string
var2 is number
var3 is object
var4 is boolean
