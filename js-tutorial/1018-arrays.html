JavaScript Arrays

JavaScript arrays are used to store multiple values in a single variable.

It is a common practice to declare arrays with the const keyword.

Creating an Array

Method 1:

Using an array literal is the easiest way to create a JavaScript Array.

Syntax: const array_name = [item1, item2, ..., itemN];

Example:
const programmingLanguages = ["HTML", "CSS", "JS"];
console.log(programmingLanguages);

Output:
HTML,CSS,JS

Method 2:

Spaces and line breaks are not important. A declaration can span multiple lines.

Example:
const programmingLanguages = [
 "HTML",
  "CSS",
 "JS"
];
console.log(programmingLanguages);

Output:
HTML,CSS,JS

Method 3:

You can also create an array, and then provide the elements.

Example:
const programmingLanguages = [];
programmingLanguages[0] = "HTML";
programmingLanguages[1] = "CSS";
programmingLanguages[2] = "JS";
console.log(programmingLanguages);

Output:
HTML,CSS,JS

Method 4:

Using the JavaScript Keyword new.

Example: const programmingLanguages = new Array("HTML", "CSS", "JS");
console.log(programmingLanguages);

Output:
HTML,CSS,JS

Accessing Array Elements

You access an array element by referring to the index number.

Array indexes start with 0.

Example:
const programmingLanguages = ["HTML", "CSS", "JS"];
let favLanguage = programmingLanguages[2];
console.log(favLanguage);

Output:
JS

Arrays are Objects

Arrays are a special type of objects. The typeof operator in JavaScript returns "object" for arrays.

Example:
const companyDetails = {name:"i-Dizital Software Solutions", location:"Australia", employees:10}; console.log(companyDetails.name); console.log(companyDetails.location); console.log(companyDetails.employees);

The Difference Between Arrays and Objects

In JavaScript, arrays use numbered indexes.

In JavaScript, objects use named indexes.

The length Property

The length property of an array returns the length of an array (the number of array elements).

The length property is always one more than the highest array index.

Adding Array Elements

Pushing

The easiest way to add a new element to an array is using the push() method.

Example:
const programmingLanguages = ["HTML", "CSS", "JS"];
programmingLanguages.push("Python");
console.log(programmingLanguages);

Output:
HTML,CSS,JS,Python

Adding elements with high indexes can create undefined "holes" in an array.

Example:
const programmingLanguages = ["HTML", "CSS", "JS", "Python"];
programmingLanguages[7] = "ReactJS";
console.log(programmingLanguages);

Output:
HTML,CSS,JS,Python,undefined,undefined,undefined,ReactJS

The push() method returns the new array length.

Example:
const programmingLanguages = ["HTML", "CSS", "JS"];
let n = programmingLanguages.push("Python");
console.log(n);
console.log(programmingLanguages);

Output:
4
HTML,CSS,JS,Python

Removing Array Elements

Popping

The pop() method removes the last element from an array.

Example:
const programmingLanguages = ["HTML", "CSS", "JS", "Python"];
programmingLanguages.pop();
console.log(programmingLanguages);

Output:
HTML,CSS,JS

The pop() method returns the value that was "popped out".

Example:
const programmingLanguages = ["HTML", "CSS", "JS", "Python"];
let popItem = programmingLanguages.pop();
console.log(popItem);
console.log(programmingLanguages);

Output:
Python
HTML,CSS,JS

Shifting Elements

Shifting is equivalent to popping, working on the first element instead of the last.

The shift() method removes the first array element and "shifts" all other elements to a lower index.

The shift() method returns the value that was "shifted out".

Unshifting Elements

The unshift() method adds a new element to an array (at the beginning), and "unshifts" older elements

The unshift() method returns the new array length.
