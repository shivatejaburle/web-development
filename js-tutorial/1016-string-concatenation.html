String Concatenation

Method 1: Using the + Operator

The same + operator you use for adding two numbers can be used to concatenate two strings.

Example:
let name = "Shiva";
let age = 18;
console.log(name + " is my name and age is " + age);

Output:
Shiva is my name and age is 18

Method 2: Using the concat() method

The concat() method joins two or more strings.

concat() does not change the existing strings, but returns a new string.

Syntax: string.concat(string1, string2, ..., stringN)

Example:
let firstName ="Shivateja";
let lastName = "Burle";
let space = " ";
let fullName = firstName.concat(space, lastName);
console.log("Full Name: " + fullName);

Output:
Full Name: Shivateja Burle

Method 3: Using Template Literals

With Template Literals, you can write exactly how you want your string to appear. So it's very easy to spot if a space is missing.

Example: let name = "Shiva";
let age = 18;
console.log(`${name} is my name and age is ${age}`);

Output:
Shiva is my name and age is 18
