Terminating Decimals and Non-Termainating Decimals

Terminating Decimal

A terminating decimal is a decimal, that has an end digit, it is a decimal, which has a finite number of digits

Example: 0.15, 0.26 etc

Non-Terminating Decimal

Non terminating decimals are the one that does not have an end digit, it has an infinite number of digits

Example: 0.5444444.... 0.1111111.... etc

Repeating and Non-Repeating Decimals

Repeating Decimal

Repeating decimals are the one, which has a set of terms in decimal to be repeated uniformly

Example: 0.666666...., 0.123123123.... etc

Repeated terms in decimals is represented by a bar on top of the repeated part

Example: 0.333333...., 0.3 bar

Non-Repeating Decimal

Non-repeating decimals are the one than does have repeated terms.

Example: 0.1010010... 28.1212211133... etc
