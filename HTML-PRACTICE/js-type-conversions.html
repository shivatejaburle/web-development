js type conversions

Converting strings to numbers

  1. Number(): Number method is used to convert a string to a number and returns a number
  2. parseInt(): parseInt method is used to parse a string and returns an integer
  3. parseFloat(): parseFloat method is used to parse a string and returns a floating point number

Converting numbers to strings

  1. String() or toString(): String method is used to convert a number to a string and returns a string
  2. toFixed(): Fixed method returns a string of number rounded with specified number of decimals
