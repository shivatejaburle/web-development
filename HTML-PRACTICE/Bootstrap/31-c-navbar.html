Navbar

Navbar comes with built-in support for a handful sub-components.

  1. .navbar-brand : used for your company name or logo
  2. .navbar-nav : used for full height navigation
  3. .navbar-toggler : used for collapse
  4. .navbar-text : used for adding text
  5. .collapse.navbar-collapse : used for grouping and hiding navbar contents
  6. .navbar-scroll : used to set max-height
