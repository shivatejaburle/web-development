Spacing

  1. Margin: Outside the border Spacing
  2. Padding: Inside the border Spacing

{property}{sides}-{Breakpoint}-{size}

property

  1. m - margin
  2. p - padding

sides

  1. t - top
  2. b - bottom
  3. e - end or right
  4. s - start or left
  5. x - left and right or start and end
  6. y - top and bottom
  7. blank - for setting all 4 sides of the element

size

  1. 0 - to set margin or padding to 0
  2. 1 - to set margin or padding to 0.25
  3. 2 - to set margin or padding to 0.5
  4. 3 - to set margin or padding to 1.0
  5. 4 - to set margin or padding to 1.5
  6. 5 - to set margin or padding to 3
  7. auto - to set margin automatically
