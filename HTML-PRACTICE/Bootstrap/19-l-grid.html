Grid System Rules

  1. Grid system allows up to 12 columns across the page.
  2. Rows must be placed in a .container(fixed-width) or .container-fluid(full-width) for proper alignment and padding.
  3. Use rows to create horizontal group of columns.
  4. Content should be placed within columns and columns may be the immediate children of rows.
