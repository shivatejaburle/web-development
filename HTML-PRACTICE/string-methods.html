String Methods

  1. String Length: Returns the Length of the String
  2. Uppercase: Returns the Capital Letters or Uppercase String
  3. Lowercase: Returns the Small Letters or Lowercase String
  4. Slice: Extract the part of the string and returns Extracted part as new String
  5. Substring: substring() is same as slice() but it doesn't negative index
  6. Substr: substr() is same as slice() but it takes the second parameter as length of the String.
  7. Search: Searches a sting and returns a start position of a match String
  8. Replace: Replaces the specified value with another value in a String
  9. Trim: Removes the spaces from both sides of a String
  10. Split: Using split it will convert to from string to array
