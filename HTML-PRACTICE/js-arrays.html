JS arrays

arrays are used to store multiple values of same type in a single variable.

syntax

const arrayName = ["Item 1", "Item 2", "Item 3", ".......", "ItemN"];

Example

const transport = ["Cars", "Buses", "Trucks", "Trains", "Light Rails", "Aeroplanes", "Helicopters", "Ships", "Jeeps", "Vans"];

Accessing array elements

You can access an array item by the refering to index number

Push()

Using Push() we can add items to the array

The Push() will return to the new array length

Pop()

Using Pop() we can remove items to the array

The Pop() Will return the value that was removed
