Favicon

A favicon, also known as a shortcut icon, website icon, tab icon, URL icon, or bookmark icon, is a file containing one or more small icons, associated with a particular website or web page.

No need to use the Favicon format as .ico we can use .png or .jpg or .jpeg formats also.

To add the Favicon add the following code in the <head> element:

 <link rel="shortcut icon" href="--path of the image--" type="image/--type of image--">
