HTML ID's

The HTML id attribute is used to specify a unique id for an HTML element.

You cannot have more than one element with the same id in an HTML document.

To create a id: write a hash (#) character, followed by a id name. Then, define the CSS properties within curly braces {}.

     #idName
     {
       property : value;
     }

     <elementName id="--your-id-name--"> - - - Your Content - - - </elementName>

Example:

#testingId
{
 border: 15px solid red;
 padding: 40px;
 text-align: center;
 font-size: x-large;
}
