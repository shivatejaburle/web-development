Web Fonts

Web fonts allow Web designers to use fonts that are not installed on the user's computer.

To use the font for an HTML element, refer to the name of the font through the font-family property.

Steps to use Google Web Fonts

  • Navigate to Google Fonts
  • Choose the Font that you like
  • Click on Select this Style
  • Click on View your selected families
  • Copy the lines in To embed a font, copy the code into the <head> of your html with link option
  • Copy the lines in CSS rules to specify families to use in your HTML Pages

Example

Inserted the following code in <head> element

<link rel="preconnect" href="https://fonts.googleapis.com">
<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>
<link href="https://fonts.googleapis.com/css2?family=Pacifico&display=swap" rel="stylesheet">

Adding Font Family

.webFont { font-family: 'Pacifico', cursive; font-size: 50px; }

Applying the Web Font

<div class="webFont"> - - - Your - - Content - - - </div>
