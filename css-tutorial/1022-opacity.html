How to make elements see-through?

Opacity - Make Element See Through

CSS opacity Property

The opacity property sets the opacity level for an element.

The opacity-level describes the transparency-level, where 1 is not transparent at all, 0.5 is 50% see-through, and 0 is completely transparent.

Examples:

Making entire block element with Opacity

#box { display: block; width: 200px; height: 200px; margin: 50px auto; padding: 25px; text-align: center; background-color: black; color: white; opacity: 0.5; }

User RGBA Values to make only background with Opacity

#box1 { display: block; width: 200px; height: 200px; margin: 50px auto; padding: 25px; text-align: center; background-color:rgba(0, 0, 0, 0.5); color: white; }
