CSS float Property

The float property specifies whether an element should float to the left, right, or not at all.

Absolutely positioned elements ignore the float property!

Syntax

float: none | left | right | initial | inherit;

  • none: The element does not float, (will be displayed just where it occurs in the text). This is default

  • left: The element floats to the left of its container.

  • right: The element floats the right of its container.

  • initial: Sets this property to its default value.

  • inherit: Inherits this property from its parent element.

Example:

.floating-image { float: left; width: 300px; margin: 10px; border: 1px solid black; box-shadow: 0 0 5px black; }

p, div { font-size: 25px; text-align: justify; }
