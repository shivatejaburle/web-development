Changing the default Width of the element

CSS width Property

The width property sets the width of an element.

The width of an element does not include padding, borders, or margins!

Examples 1:

div.half-width { font-size: 20px; border: 8px solid red; padding: 10px; width: 48%; display: inline-block; }
