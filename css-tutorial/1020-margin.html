Margin

The External Element Spacing

Syntax

 margin: top right bottom left;

You can use margin-top, margin-right, margin-bottom and margin-left separately to get individual borders.

This property can have from one to four values.

If the margin property has four values:
  • margin:10px 5px 15px 20px;
    • top margin is 10px
    • right margin is 5px
    • bottom margin is 15px
    • left margin is 20px
If the margin property has three values:
  • margin:10px 5px 15px;
    • top margin is 10px
    • right and left margin are 5px
    • bottom margin is 15px
If the margin property has two values:
  • margin:10px 5px;
    • top and bottom margin are 10px
    • right and left margin are 5px
If the margin property has one value:
  • margin:10px;
    • all four margins are 10px

Examples:

#box { width: 200px; height: 200px; border: 1px dashed red; margin: 20px; }
