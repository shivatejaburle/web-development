Pseudo Classes

A pseudo-class is used to define a special state of an element.

Using pseudo-class we can:

  • Style an element when a user mouses over it
  • Style visited and unvisited links differently
  • Style an element when it gets focus

Syntax

selector:pseudo-class
{
 property: value;
}

Example

.box { height: 30px; border: 1px solid blue; color: white; }
.box:not(.doNotSelectMe) { background-color: red; }
.box:hover { background-color: grey; }
.box:nth-child(6) { background-color: black; }
