Padding

The Internal Element Spacing i.e., Padding creates extra space within an element.

An element's padding is the space between its content and its border.

Syntax

 padding: top right bottom left;

You can use padding-top, padding-right, padding-bottom and padding-left separately to get individual borders.

This property can have from one to four values.

If the padding property has four values:
  • padding:10px 5px 15px 20px;
    • top padding is 10px
    • right padding is 5px
    • bottom padding is 15px
    • left padding is 20px
If the padding property has three values:
  • padding:10px 5px 15px;
    • top padding is 10px
    • right and left padding are 5px
    • bottom padding is 15px
If the padding property has two values:
  • padding:10px 5px;
    • top and bottom padding are 10px
    • right and left padding are 5px
If the padding property has one value:
  • padding:10px;
    • all four paddings are 10px

Example:

#box { width: 200px; height: 200px; border: 1px dashed red; padding: 20px; }
