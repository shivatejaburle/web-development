Fixed Positioning (like headers)

The element is positioned relative to the browser window.

Example:

body { margin: 0; padding: 0; height: 3000px; }
p { padding: 10px; margin: 20px; font-size: 30px; }
.box { background-color: #f5f5f5; border-bottom: 2px solid #CCC; padding: 20px; position: fixed; top: 0; left: 0; width: 100%; }
.content { margin-top: 150px; }

<div class="box"> It is Fixed Header </div>
<div class="content"> It is Content </div>
