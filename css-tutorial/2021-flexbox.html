FlexBox

Before the Flexbox Layout module, there were four layout modes:

  1. Block - for sections in a webpage
  2. Inline - for text
  3. Table - for two-dimensional table data
  4. Positioned - for explicit position of an element

The Flexible Box Layout Module, makes it easier to design flexible responsive layout structure without using float or positioning.

Flexbox Elements

To start using the Flexbox model, you need to first define a flex container.

Example:

#container1 { display: flex; background-color: orange; }
#container1 div { border: 1px solid red; padding: 10px; margin: 10px; }

<div id="container1">
 <div>Flexbox 1</div>
 <div>Flexbox 2</div>
 <div>Flexbox 3</div>
 <div>Flexbox 4</div>
 <div>Flexbox 5</div>
 <div>Flexbox 6</div>
</div>

flex-flow

The flex-flow property is a shorthand property for setting both the flex-direction and flex-wrap properties.

#container { display: flex; flex-flow: row wrap; }

Flex Items

The direct child elements of a flex container automatically becomes flexible (flex) items.

The flex item properties are:

The order property specifies the order of the flex items.

#container7 div { border: 1px solid black; padding: 10px; margin: 10px; color: black; }

<div id="container7">
 <div style="order: 3">Flexbox 1</div>
 <div style="order: 2">Flexbox 2</div>
 <div style="order: 1">Flexbox 3</div>
 <div style="order: 6">Flexbox 4</div>
 <div style="order: 4">Flexbox 5</div>
 <div style="order: 5">Flexbox 6</div>
</div>

The align-self property specifies the alignment for the selected item inside the flexible container.

Responsive Flexbox

You can use media queries to create different layouts for different screen sizes and devices.

Example:

For example, if you want to create a two-column layout for most screen sizes, and a one-column layout for small screen sizes (such as phones and tablets), you can change the flex-direction from row to column at a specific breakpoint (800px in the example below).

#container12 { display: flex; flex-direction: row; background-color: powderblue; }
#container12 div { border: 1px solid black; padding: 10px; margin: 10px; color: black; }
.left-content { background-color: red; padding: 10px; flex: 50%; }
.right-content { background-color: yellow; padding: 10px; flex: 50%; }
@media (max-width: 800px)
{
  #container12 { flex-direction: column; }
}

<div id="container12">
 <div class="left-content">Flexbox 1</div>
 <div class="right-content">Flexbox 2</div>
</div>
