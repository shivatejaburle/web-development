Background Image

The background-image property sets background images for an element.

Syntax

background-color : url ('- - path of the image - -')

CSS Properties for Background Image:

  • background-position : center | left | right | top | bottom
  • background-repeat : repeat | repeat-x | repeat-y | no-repeat
  • background-size : auto | length | cover | contain

Example:

div
{
 background-image: url('images/web-development.png');
 background-size: contain;
 background-repeat: no-repeat;
 background-position: right;
}
