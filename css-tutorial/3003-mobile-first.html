Always Design for Mobile First

Mobile First means designing for mobile before designing for desktop or any other device (This will make the page display faster on smaller devices).

This means that we must make some changes in our CSS.

Instead of changing styles when the width gets smaller than 768px, we should change the design when the width gets larger than 768px. This will make our design Mobile First.

resbox:

body
{
 background-color: lightcoral;
}

/* breakpoint at 640px */
@media (min-width:640px)
{
 body
 {
  background-color: aqua;
 }
}

Device Breakpoints

There are tons of screens and devices with different heights and widths, so it is hard to create an exact breakpoint for each device.

To keep things simple you could target five groups.

  1. Extra small devices (phones, 600px and down)
  2. Small devices (portrait tablets and large phones, 600px and up)
  3. Medium devices (landscape tablets, 768px and up)
  4. Large devices (laptops/desktops, 992px and up)
  5. Extra large devices (large laptops and desktops, 1200px and up)

Example:

Resize your browser to change the color

.resbox { padding: 20px; color: white; }
@media only screen and (max-width: 600px) { .resbox {background: red;} }
@media only screen and (min-width: 600px) { .resbox {background: green;} }
@media only screen and (min-width: 768px) { .resbox {background: blue;} }
@media only screen and (min-width: 992px) { .resbox {background: orange;} }
@media only screen and (min-width: 1200px) { .resbox {background: black;} }

<div class="resbox">
 <p>Resize your browser to change the color</p>
</div>
