Sticky Positioning (like modern headers)

Explanation and Example

The element is positioned based on the user's scroll position.

A sticky element toggles between relative and fixed, depending on the scroll position. It is positioned relative until a given offset position is met in the viewport - then it "sticks" in place (like position:fixed).

Example:

body { margin: 0; padding: 0; }
p { padding: 10px; margin: 20px; font-size: 30px; }
.box { background-color: red; border-bottom: 2px solid #CCC; padding: 20px; color: black; font-size: 30px; position: Sticky; top: 0; }

<h1> --- Main Heading --- </h1>
<div style="background-color: green; color: white;">
 <div class="box"> --- Section Heading --- </div>
   --- Your Section Content ---
</div>
<div style="background-color: blue; color: white;">
 <div class="box"> --- Section Heading ---</div>
   --- Your Section Content ---
 </div>
</div>
