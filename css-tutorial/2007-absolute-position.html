Absolute Positioning

The element is positioned relative to its first positioned (not static) ancestor element.

The parent element must be relative when you want to use Absolute Position.

Example:

.box { width: 300px; height: 300px; border: 4px solid red; position: relative; }
.second-box { position: absolute; border: 3px solid blue; top: 0; left: 0; width: 45%; height: 45%; }
.third-box { position: absolute; border: 3px solid blue; top: 0; right: 0; width: 45%; height: 45%; }
.fourth-box { position: absolute; border: 3px solid blue; bottom: 0; left: 0; width: 45%; height: 45%; }
.fifth-box { position: absolute; border: 3px solid blue; bottom: 0; right: 0; width: 45%; height: 45%; }

<div class="box">
  <div class="second-box"> Top Left </div>
  <div class="third-box"> Top Right </div>
  <div class="fourth-box"> Bottom Left </div>
  <div class="fifth-box"> Bottom Right </div>
</div>
