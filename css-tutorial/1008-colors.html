CSS Colors

Colors are specified using Predefined Color Names, or RGB, HEX, HSL, RGBA, HSLA values.

  • HEX value

    • In CSS, a color can be specified using a hexadecimal value as #RRGGBB
    • Where rr (red), gg (green) and bb (blue) are hexadecimal values between 00 and ff (same as decimal 0-255)
    • Eg: #92a8d1

  • RGB value

    • In CSS, a color can be specified as an RGB value as rgb(red, green, blue)
    • Each parameter (red, green, and blue) defines the intensity of the color between 0 and 255
    • Eg: rgb(201, 76, 76)

  • RGBA value

    • RGBA color values are an extension of RGB color values with an alpha channel (which specifies the opacity for a color) as rgba(red, green, blue, alpha)
    • The alpha parameter is a number between 0.0 (fully transparent) and 1.0 (not transparent at all)
    • Eg: rgba(201, 76, 76, 0.6)

  • HSL value

    • In CSS, a color can be specified using hue, saturation, and lightness (HSL) in the form as hsl(hue, saturation, lightness)
    • Hue is a degree on the color wheel from 0 to 360. 0 is red, 120 is green, and 240 is blue.
    • Saturation is a percentage value, 0% means a shade of gray, and 100% is the full color.
    • Lightness is also a percentage, 0% is black, 50% is neither light or dark, 100% is white
    • Eg: hsl(89, 43%, 51%)

  • HSLA value

    • HSLA color values are an extension of HSL color values with an alpha channel (which specifies the opacity for a color) as hsla(hue, saturation, lightness, alpha)
    • The alpha parameter is a number between 0.0 (fully transparent) and 1.0 (not transparent at all).
    • Eg: hsla(89, 43%, 51%, 0.6)
