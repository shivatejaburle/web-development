Adding Box Shadows for eye-popping element

CSS box-shadow Property

The box-shadow property attaches one or more shadows to an element.

Syntax

box-shadow: none|h-offset v-offset blur spread color |inset|initial|inherit;

  • none: Default value. No shadow is displayed
  • h-offset: Required. The horizontal offset of the shadow. A positive value puts the shadow on the right side of the box, a negative value puts the shadow on the left side of the box
  • v-offset: Required. The vertical offset of the shadow. A positive value puts the shadow below the box, a negative value puts the shadow above the box
  • blur: Optional. The blur radius. The higher the number, the more blurred the shadow will be
  • spread: Optional. The spread radius. A positive value increases the size of the shadow, a negative value decreases the size of the shadow
  • blur: Optional. The color of the shadow. The default value is the text color. Look at CSS Color Values for a complete list of possible color values.
  • inset: Optional. Changes the shadow from an outer shadow (outset) to an inner shadow
  • initial: Sets this property to its default value.
  • inherit: Inherits this property from its parent element.

Examples:

#rectangle { border: 1px solid; padding: 10px; width: 300px; height: 200px; box-shadow: 10px 10px; }

#shadowColor { border: 1px solid; padding: 10px; width: 300px; height: 200px; border-radius: 10%; box-shadow: 10px 10px red; }

#innerShadow { border: 1px solid; padding: 10px; width: 300px; height: 200px; border-radius: 10%; box-shadow: 10px 10px inset red; }

#blurShadow { border: 1px solid; padding: 10px; width: 300px; height: 200px; border-radius: 0 10% 10% 0; box-shadow: 10px 10px 20px inset red; }

#profilePicture { border: 10px solid orange; border-radius: 50%; height: 200px; width: 200px; background-image: url('images/profile-pic.png'); background-size: cover; box-shadow: -10px -10px 20px grey; }

#box { width: 400px; height: 600px; border: 5px solid yellowgreen; background-image: url('images/garden-walkway.jpg'); background-position: center center; background-size: cover; box-shadow: 6px 6px 15px inset red; }



#leaf { width: 400px; height: 600px; border: 5px solid yellowgreen; background-image: url('images/garden-walkway.jpg'); background-position: center center; background-size: cover; border-radius: 0 50% 0 50%; box-shadow: 6px 6px 15px inset red; }
