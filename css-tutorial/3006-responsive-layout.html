Responsive Layout

Responsive design is an approach to web page creation that makes use of flexible layouts, flexible images and cascading style sheet media queries. The goal of responsive design is to build web pages that detect the visitor's screen size and orientation and change the layout accordingly.

Example

CSS

* { box-sizing: border-box; }
body { padding: 0; margin: 0; }
#layout { display: flex; flex-wrap: wrap; }
#nav { flex-basis: 100%; border-bottom: 1px solid grey; padding: 20px; }
#sidebar { flex-basis: 20%; border-right: 1px solid gray; padding: 20px; border-bottom: 1px solid grey; }
#content { flex-basis: 80%; padding: 20px; border-bottom: 1px solid grey; }
@media (max-width:640px)
{
  #sidebar { flex-basis: 100%; border-right: 0; }
  #content { flex-basis: 100%; border-bottom: 0; }
}
@media (max-width:475px)
{
  #sidebar { display: none; }
}

HTML

<div id="nav">
  <strong>Site Navigation</strong>
</div>

  <strong>Site Sidebar</strong>
</div>
<div id="content">

</div>
