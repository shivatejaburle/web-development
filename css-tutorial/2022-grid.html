GRID

Grid Layout

The CSS Grid Layout Module offers a grid-based layout system, with rows and columns, making it easier to design web pages without having to use floats and positioning.

Grid Container

To make an HTML element behave as a grid container, you have to set the display property to grid or inline-grid.

Grid containers consist of grid items, placed inside columns and rows.

Grid Elements

A grid layout consists of a parent element, with one or more child elements.

Display Property

An HTML element becomes a grid container when its display property is set to grid or inline-grid.

  • Set the display property to grid to make a block-level grid container.
  • Set the display property to inline-grid to make an inline grid container.

All direct children of the grid container automatically become grid items.

grid-template-columns Property

The grid-template-columns property defines the number of columns in your grid layout, and it can define the width of each column.

The value is a space-separated-list, where each value defines the width of the respective column.

Example: If you want your grid layout to contain 4 columns, specify the width of the 4 columns, or "auto" if all columns should have the same width.

Note: If you have more than 4 items in a 4 columns grid, the grid will automatically add a new row to put the items in.

The grid-template-columns property can also be used to specify the size (width) of the columns.

Example for Grid with display property as grid:

#gridStyle1 { display: grid; grid-template-columns: auto auto auto; }
.itemStyle1 { border: 1px solid red; }

Grid Container Properties

  • grid-template-columns
    • The grid-template-columns property defines the width of each row.
    • The value is a space-separated-list, where each value defines the width of the respective column.
    • Values:
      • auto
      • specify required width
  • grid-template-rows
    • The grid-template-rows property defines the height of each row.
    • The value is a space-separated-list, where each value defines the height of the respective row.
    • Values:
      • auto
      • specify required width
  • justify-content
    • The justify-content property is used to align the whole grid inside the container.
    • Note: The grid's total width has to be less than the container's width for the justify-content property to have any effect.
    • Values:
      • space-evenly
      • space-around
      • space-between
      • center
      • start
      • end
  • align-content
    • The align-content property is used to vertically align the whole grid inside the container.
    • Note: he grid's total height has to be less than the container's height for the align-content property to have any effect.
    • Values:
      • space-evenly
      • space-around
      • space-between
      • center
      • start
      • end

Grid Item Properties

  • grid-column
    • The grid-column property defines on which column(s) to place an item.
    • You define where the item will start, and where the item will end.
    • The grid-column property is a shorthand property for the grid-column-start and the grid-column-end properties.
    • To place an item, you can refer to line numbers, or use the keyword "span" to define how many columns the item will span.

      • Example:

      #gridStyle6 { display: grid; grid-template-columns: auto auto auto auto; grid-gap: 20px; }
      .itemStyle6 { border: 1px solid red; padding: 10px; background-color: gold; }

      /* Make Grid Item start on column 1 & end before column 3 */
      .merge-cols-1 { grid-column: 1 / 3; }

      /* Make Grid Item start on column 3 and span 2 columns */
      .merge-cols-2 { grid-column: 3 / 5; }

      <div id="gridStyle6">
       <div class="itemStyle6 merge-cols-1">Item 1</div>
       <div class="itemStyle6">Item 2</div>
       <div class="itemStyle6">Item 3</div>
       <div class="itemStyle6">Item 4</div>
       <div class="itemStyle6">Item 5</div>
       <div class="itemStyle6 merge-cols-2">Item 6</div>

  • Naming Grid Items
    • The grid-area property can also be used to assign names to grid items.
    • Named grid items can be referred to by the grid-template-areas property of the grid container.
    • Each row is defined by apostrophes (' ')
    • The columns in each row is defined inside the apostrophes, separated by a space.
    • Note: A period sign represents a grid item with no name.

      • Example:

      .myGridArea { grid-area: area1; }
      #gridStyle9 { display: grid; grid-template-areas: 'area1 . . .' 'area1 . . .'; grid-gap: 20px; }
      .itemStyle9 { border: 1px solid red; padding: 10px; background-color: cornflowerblue; }

      <div id="gridStyle9">

  • Order of the Items
    • The Grid Layout allows us to position the items anywhere we like.
    • The first item in the HTML code does not have to appear as the first item in the grid.
    • You can re-arrange the order for certain screen sizes, by using media queries.

      • Example:

      .gridStyle10 { display: grid; grid-template-columns: auto auto auto; grid-gap: 20px; }
      .gridStyle10 > div { border: 1px solid red; padding: 10px; background-color: powderblue; }

      .order1 { grid-area: 1 / 3 / 2 / 4; }
      .order2 { grid-area: 2 / 3 / 3 / 4; }
      .order3 { grid-area: 1 / 1 / 2 / 2; }
      .order4 { grid-area: 1 / 2 / 2 / 3; }
      .order5 { grid-area: 2 / 1 / 3 / 2; }
      .order6 { grid-area: 2 / 2 / 3 / 3; }

      @media only screen and (max-width: 800px)
      {
       .order1 { grid-area: 1 / span 3 / 2 / 4; }
       .order2 { grid-area: 3 / 3 / 4 / 4; }
       .order3 { grid-area: 2 / 1 / 3 / 2; }
       .order4 { grid-area: 2 / 2 / span 2 / 3; }
       .order5 { grid-area: 3 / 1 / 4 / 2; }
       .order6 { grid-area: 2 / 3 / 3 / 4; }
      }
