CSS Element Selectors

The element selector selects HTML elements based on the element name.

Example:
body { background-color: aqua; }
h1 { color: brown; }
div { background-color: magenta; font-size: 25px; padding: 10px; }
p { background-color: darkmagenta; color: white; font-size: 20px; padding: 5px; }
