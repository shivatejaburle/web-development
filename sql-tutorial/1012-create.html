Creating a New Table

The CREATE TABLE statement is used to create a new table in a database.

Syntax

 CREATE TABLE table_name (
  column1 datatype,
  column2 datatype,
  column3 datatype,
   ....
 );

Example:

 create table mystore(
  id integer,
  item_name VARCHAR(50),
  price decimal(5,2),
  notes VARCHAR(255),
  items_sold INTEGER DEFAULT 0
 );

Check the table:

SELECT * FROM mystore;

Insert data to the table:

INSERT INTO mystore (id, item_name, price, notes) values (1, 'Fidget Spinner', 1.99, 'It is so colorful!');
